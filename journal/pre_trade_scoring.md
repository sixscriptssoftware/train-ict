# ICT Trade Scoring System

## How to Use:
Before taking any trade, score it against these criteria. Aim for 7+/10.

---

## üìä SCORING CRITERIA (10 Points Max)

### HTF Alignment (3 Points)
| Score | Criteria |
|-------|----------|
| 3 | Daily bias clear + Weekly structure supports + Monthly draw visible |
| 2 | Daily bias clear + One higher TF supports |
| 1 | Daily bias established |
| 0 | No clear HTF bias |

### Model Clarity (2 Points)
| Score | Criteria |
|-------|----------|
| 2 | Trading exact ICT model (Buy/Sell Model, Silver Bullet, Judas, etc.) |
| 1 | Partial model elements present |
| 0 | No clear model |

### Session/Timing (2 Points)  
| Score | Criteria |
|-------|----------|
| 2 | Inside prime killzone (London Open, NY AM) + favorable day (Tue-Thu) |
| 1 | Inside any killzone OR favorable day |
| 0 | Outside killzone on unfavorable day |

### Confluence Count (2 Points)
| Score | Criteria |
|-------|----------|
| 2 | 5+ confluences |
| 1 | 3-4 confluences |
| 0 | Less than 3 confluences |

### Risk:Reward (1 Point)
| Score | Criteria |
|-------|----------|
| 1 | 3:1 or better with clear target |
| 0 | Less than 3:1 or unclear target |

---

## üéØ SCORE INTERPRETATION

| Score | Action |
|-------|--------|
| 9-10 | **A+ SETUP** - Full size, high confidence |
| 7-8 | **GOOD SETUP** - Standard size |
| 5-6 | **MARGINAL** - Reduce size or pass |
| 0-4 | **DO NOT TRADE** - Missing too many elements |

---

## ‚úÖ QUICK CHECKLIST (Must-Haves)

Before ANY trade, confirm:
- [ ] HTF bias is clear (not guessing)
- [ ] I can name the ICT model
- [ ] I'm in or near a killzone
- [ ] Stop loss is at a level that makes sense (not arbitrary)
- [ ] Target is based on liquidity (not just "hope")
- [ ] Risk is sized correctly for account
- [ ] I'm not revenge trading
